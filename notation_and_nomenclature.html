Kinematics: Notation and Nomenclature

Notation

Scalars are represented as italic, e.g., \(b\). Vectors are represented as bold lower case, e.g., \(\mathbf{x}\). The scalar components of a vector are denoted by the corresponding italic symbol with a single numerical index , i.e., given \(\mathbf{x} \in \mathbb{R}^{n} \), the components are \( \mathbf{x} = \left[ \begin{array}{cccc} x_1 & x_2& \cdots & x_n \end{array} \right]^T \). Given \(\mathbf{x}^{\gamma}_{\beta \alpha} \), it descibes a kinematic property of frame \(\alpha\) with respect to frame \(\beta\) expressed in the frame \(\gamma\) axes.
Matrices are represented as bold upper case, e.g., transformation matrix \(\mathbf{T}\). A matrix denoted as \(R^{\beta}_{\alpha}\) denotes a rotation matrix from frame \(\alpha\) to frame \(\beta\). When a matrix is used to represent an attitude, it is more common to use the upper index to represent the reference frame and the lower index to represent the object frame. More information can be found in active and passive transformation. Finally, all coordinate frames are in right-handed convention.
